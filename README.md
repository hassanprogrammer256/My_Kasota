## Features
- Classic snake movement and growth mechanics
- Score tracking
- Pause functionality
- Game over screen with restart option
- Grid overlay for better visibility
- Smooth controls with direction change prevention
- Authentication and Authorization functionality with PIN entry

## How to Play
1. Run `python main.py`
2. Provide your PIN to open the Game (default: 1234)
3. Use arrow keys to control the snake
4. Eat red food to grow and increase score
5. Avoid hitting walls or yourself
6. Press 'ESC' to pause/unpause
7. Press 'SPACE' to restart after game over

## Controls
- **Arrow Keys**: Change snake direction
- **ESC**: Pause/Unpause game
- **ESC** (during game over): Quit game
- **SPACE**: Restart after game over

## Requirements
- Python 3.x
- Pygame library
- Pygwidgets library
- Python-dotenv (for PIN configuration)
