# Snake Game

A classic Snake game implemented in Python using Pygame and pyqwidgets.

## Features
- Classic snake movement and growth mechanics
- Score tracking
- Pause functionality
- Game over screen with restart option
- Grid overlay for better visibility
- Smooth controls with direction change prevention
- Authentication and Authorization functionality

## How to Play
1. Run `python main.py`
2. provide your PIN to open the Game
3. Use arrow keys to control the snake
4. Eat red food to grow and increase score
5. Avoid hitting walls or yourself
6. Press 'SPACE' to pause/unpause
7. Press 'ESC' to quit

## Controls
- **Arrow Keys**: Change snake direction
- **P**: Pause/Unpause game
- **ESC**: Quit game
- **SPACE**: Restart after game over


## Requirements
- Python 3.x
-Pygwidgets
- Pygame library
